# PCL CE下载

## 最新稳定版

### 系统架构检查

按下 **Win + S** 键打开搜索，搜索 **系统信息** 并打开。

查看 **系统类型**：
- 若显示 **x64 电脑** → 下载 X64 版本
- 若显示 **ARM64 电脑** → 下载 ARM64 版本

::: tip 小提示
- 常规台式机/笔记本通常为 **X64**
- **ARM64** 主要用于微软 Surface Pro X 等设备
- 若不确定，优先选 **X64**（兼容性更广），若不可用再选 ARM64
:::
