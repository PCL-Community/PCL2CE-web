# PCL CE Download

## Latest Stable Release

### System Architecture Check

Press **Win + S** to open search, search for **System Information** and open it.

Check **System Type**:
- If it shows **x64-based PC** → Download X64 version
- If it shows **ARM64-based PC** → Download ARM64 version

::: tip Tips
- Regular desktops/laptops are usually **X64**
- **ARM64** is mainly used for devices like Microsoft Surface Pro X
- If unsure, choose **X64** first (wider compatibility), try ARM64 if unavailable
:::
