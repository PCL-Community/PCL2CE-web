Press Win + S to open Search, search for System Information and open it.

View system type:

If it shows x64 PC → Download the X64 version.

If it shows ARM64 PC → Download the ARM64 version.

💡 Tips:

Regular desktops/laptops are usually X64.

ARM64 is mainly used in the ARM64 version.

ARM64 is mainly used in devices like Microsoft Surface Pro X.

If you're not sure, prioritize X64 (for wider compatibility), and then ARM64 if it's not available.
